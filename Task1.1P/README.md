# Task 1.1P – Switching ON Lights

This project simulates a smart lighting system for an retirement village home.

## System Description

When the push button is pressed:

- The porch light turns ON for 30 seconds.
- The hallway light turns ON for 60 seconds.

This helps Linda safely enter her house at night.

## Modular Programming Implementation

The program uses modular programming by dividing the system into functions:

readButton() – detects when the push button is pressed.

turnPorchLightOn() – activates the porch LED and starts a timer.

turnHallwayLightOn() – activates the hallway LED and starts a timer.

handleLights() – checks timers and turns lights OFF after the required duration.
